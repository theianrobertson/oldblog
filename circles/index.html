What's This?

How's it work?

The tool uses the fantastic Pillow library to do picture manipulations. It actually "draws" around the circles by:

  1. Creating a blank canvas the same size as the image
  2. Picking a spot in the middle of the canvas
  3. Iterating around 360 degrees, growing the radius slightly each time with some arbitrary "wobbles"
  4. Drawing lines between subsequent points, where the width of the line is proportional to the greyscale value (white would be a minimum width, black a maximum width)
  5. Repeating the 360 degree iteration a bunch of times

  7. Profit!
